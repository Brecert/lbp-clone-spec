A game similar to LittleBigPlanet(2) for mobile.

Takes inspiration from LittleBigPlanet Vita for the ui and interface.

Core

1. Platformer
    - Up, Down, Left, Right
2. Movement Options
    - Analog Left/Right
    - Analog Jump based on time held
    - Grabbing
3. Physics
    - Every object (unless specified) can be interacted with and moved
    - Bouncyness
    - Slipperyness / Friction
    - Heavyness / Weight
    - Gravity (?)
    - Timescale (?)
4. Dangers
    - Crushed
    - Spikes
    - Fire
    - Electricity
    - Ice
    - Explosion
    - Gas
5. Obstacles
    - Wind
6. Tools
    - String
    - Elastic String
    - Spring
    - Winch
7. World
    - Size
    - Gravity
    - Lighting
    - Fog
    - Background
    - Theme
    - Soundscape
    - Sounds
        - Echo
        - Distortion
